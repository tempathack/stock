<objective>
Implement the yfinance macro data fetcher and the join logic that attaches macro features to each stock's daily rows.

Four features per trading day (broadcast to all tickers for that date):
1. vix         — ^VIX daily close, forward-filled on holidays
2. spy_return  — SPY daily log-return
3. sector_return — the sector ETF log-return for the ticker's GICS sector
4. high52w_pct — (close - low_52w) / low_52w  (computed from ohlcv_daily rolling window)
5. low52w_pct  — (high_52w - close) / high_52w (computed from ohlcv_daily rolling window)

52-week high/low: compute from existing OHLCV data using 252-day rolling window:
```python
df["high52w"] = df["close"].rolling(252, min_periods=20).max()
df["low52w"]  = df["close"].rolling(252, min_periods=20).min()
df["high52w_pct"] = (df["high52w"] - df["close"]) / df["high52w"]
df["low52w_pct"]  = (df["close"] - df["low52w"])  / df["low52w"]
```
