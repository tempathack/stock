<objective>
Fix four stat panels in the Grafana API Health dashboard that display "No data" text on a green background, falsely signalling a healthy state when metrics are absent.

Purpose: Green no-data panels create alert fatigue and false confidence — operators cannot distinguish "system healthy at 0% errors" from "exporter not running, no data scraped." Making no-data states visually neutral (blue) restores the signal/noise contract.

Output: Updated `grafana-dashboard-api-health.yaml` ConfigMap with Grafana `mappings` type `special` / `null+nan` entries on panels 2, 3, 4, 5. Threshold steps are untouched. Grafana reloaded and panels verified visually via port-forward.
</objective>
