## Design Decisions
- **node:20-alpine** — LTS Node with smallest image size (matches Vite 6 requirements)
- **nginx:alpine** — ~5MB base image for static serving; much smaller than Node serving
- **SPA try_files fallback** — all routes (including /dashboard, /models, etc.) fall back to index.html for client-side routing
- **VITE_API_URL as build arg** — Vite inlines env vars at build time (not runtime); must be set during `npm run build`
- **Gzip in Nginx** — compresses JS/CSS bundles ~60-70% for faster load
- **Cache-Control split** — hashed assets get 1-year cache (immutable); index.html gets no-cache (always fresh)
- **No API proxy in Nginx by default** — K8s deployment uses direct `VITE_API_URL` env; Nginx proxy_pass is optional comment for future ingress pattern
- **npm ci preferred over npm install** — deterministic install from lockfile; falls back to npm install if no lockfile exists yet
