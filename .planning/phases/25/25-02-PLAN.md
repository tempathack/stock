## Design Decisions
- **Sidebar over top nav** — Bloomberg Terminal uses persistent sidebar for quick screen switching; more space efficient for data-heavy pages
- **Lazy-loaded pages** — each page is React.lazy() imported to keep initial bundle small; Suspense shows LoadingSpinner
- **Active NavLink styling** — left accent border + bg-bg-card matches Bloomberg's active tab pattern
- **Unicode icons over icon library** — avoids react-icons/lucide-react dep for 4 icons; simple SVGs inline if unicode is insufficient
- **PlaceholderCard component** — standardized "coming soon" pattern for Phases 26–29 skeleton pages
- **/ redirects to /dashboard** — dashboard is the home page (Bloomberg's default screen)
- **API status indicator in sidebar** — subtle UX: green dot confirms backend connectivity, red dot signals issues (uses useHealthCheck hook)
