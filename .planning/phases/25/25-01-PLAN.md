## Design Decisions
- **TypeScript over JavaScript** — type safety for API response shapes prevents runtime bugs in data-heavy dashboard
- **Tailwind v4 CSS-first config** — no `tailwind.config.js` file; `@theme` block in CSS replaces it (simpler, faster)
- **Axios over fetch** — interceptors for error logging, base URL config, request/response transforms
- **React Query over Zustand for server state** — automatic caching, refetch, loading/error states; Zustand would be overkill for read-heavy dashboard
- **Inter font** — clean sans-serif that matches Bloomberg Terminal readability; imported from Google Fonts, not bundled
- **staleTime 30s** — stock data doesn't need sub-second freshness; reduces API load
- **.jsx → .tsx rename** — existing placeholder files are .jsx; new files use .tsx. Plan 02 will replace App.jsx with App.tsx
