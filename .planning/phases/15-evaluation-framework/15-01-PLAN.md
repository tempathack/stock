## Files to Create

### 1. `ml/evaluation/ranking.py`

**Dataclasses:**

```python
@dataclass
class RankedModel:
    """A TrainingResult augmented with ranking metadata."""
    rank: int
    training_result: TrainingResult
    composite_score: float       # adjusted RMSE (lower = better)
    raw_rmse: float
    directional_accuracy: float
    fold_stability: float
    variance_penalty: float      # λ × fold_stability
```

```python
@dataclass
class WinnerResult:
    """The selected winner model with comparison context."""
    winner: RankedModel
    runner_up: RankedModel | None
    margin: float                # composite score gap to runner-up
    total_candidates: int
```

**Functions:**

```python
def compute_composite_score(
    result: TrainingResult,
    stability_penalty_weight: float = 0.5,
    directional_accuracy_weight: float = 0.01,
) -> float:
```
- Formula: `oos_rmse + λ_stab * fold_stability - λ_da * directional_accuracy`
- Additive penalty for instability, small bonus for directional accuracy
- Returns a single scalar (lower = better)

```python
def rank_models(
    results: list[TrainingResult],
    stability_penalty_weight: float = 0.5,
    directional_accuracy_weight: float = 0.01,
) -> list[RankedModel]:
```
- Compute composite score for each result
- Sort ascending (best first)
- Assign ordinal rank (1-based)
- Return list of `RankedModel`
- Raises `ValueError` if results list is empty

```python
def select_winner(
    results: list[TrainingResult],
    stability_penalty_weight: float = 0.5,
    directional_accuracy_weight: float = 0.01,
) -> WinnerResult:
```
- Calls `rank_models()` internally
- Returns `WinnerResult` with rank-1 as winner
- Logs winner name, score, margin over runner-up
- Raises `ValueError` if results list is empty

### 2. `ml/tests/test_ranking.py`

Use synthetic `TrainingResult` objects (no actual model training).

**Fixtures:**
- `sample_training_results()` — 5 `TrainingResult` objects with hand-crafted metric values covering:
  - Low RMSE + low variance (should win)
  - Low RMSE + high variance (should be penalized)
  - High RMSE + high directional accuracy
  - Medium across the board
  - Worst RMSE

**Test classes:**

`TestCompositeScore` (4 tests):
- `test_zero_penalty_equals_rmse` — with λ=0, score = raw RMSE (minus tiny DA bonus)
- `test_penalty_increases_score` — higher fold_stability → higher composite score
- `test_directional_accuracy_bonus` — higher DA → lower composite score
- `test_custom_weights` — non-default weights produce expected shift

`TestRankModels` (6 tests):
- `test_returns_ranked_models` — len matches input, all are RankedModel instances
- `test_rank_order_ascending` — ranks are 1, 2, 3, ...
- `test_best_model_first` — rank-1 has lowest composite score
- `test_high_variance_penalized` — low-RMSE-but-unstable model ranked below stable competitor
- `test_empty_results_raises` — ValueError on empty list
- `test_single_result` — single model gets rank 1

`TestSelectWinner` (5 tests):
- `test_winner_is_rank_one` — winner.rank == 1
- `test_margin_positive` — margin > 0 when multiple candidates
- `test_runner_up_exists` — runner_up is not None when ≥2 candidates
- `test_single_candidate_no_runner_up` — runner_up is None, margin is 0.0
- `test_total_candidates_count` — matches input length
