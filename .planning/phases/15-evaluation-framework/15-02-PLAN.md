## Files to Create / Update

### 1. `ml/models/registry.py` (replace stub)

**Storage layout:**
```
{base_dir}/
  {model_name}_{scaler_variant}/
    v{version}/
      pipeline.pkl          # fitted sklearn Pipeline (scaler + model)
      metadata.json         # all metrics, params, ranking info, timestamps
      features.json         # ordered list of feature column names
```

**Class: `ModelRegistry`**

```python
class ModelRegistry:
    """File-based model registry for artifact persistence and retrieval."""

    def __init__(self, base_dir: str = "model_registry"):
        ...
```

**Methods:**

```python
def save_model(
    self,
    result: TrainingResult,
    pipeline: Pipeline,
    feature_names: list[str],
    rank: int | None = None,
    is_winner: bool = False,
    version: int | None = None,  # auto-increment if None
) -> str:
```
- Creates directory `{model_name}_{scaler_variant}/v{version}/`
- Pickles pipeline to `pipeline.pkl`
- Writes `metadata.json` with: model_name, scaler_variant, best_params, oos_metrics, fold_metrics, fold_stability, rank, is_winner, saved_at (ISO timestamp), version
- Writes `features.json` with ordered feature names
- Auto-increments version by scanning existing version dirs
- Returns the artifact directory path

```python
def load_model(self, model_name: str, scaler_variant: str, version: int | None = None) -> dict:
```
- Loads latest version if version is None
- Returns dict with keys: `pipeline`, `metadata`, `feature_names`, `version`, `path`
- Raises `FileNotFoundError` if not found

```python
def list_models(self) -> list[dict]:
```
- Scans base_dir for all model/version combos
- Returns list of dicts with: model_name, scaler_variant, version, is_winner, oos_rmse, path
- Sorted by oos_rmse ascending

```python
def get_winner(self) -> dict | None:
```
- Scans all metadata.json for `is_winner: true`
- Returns the latest version with is_winner flag, or None

```python
def delete_model(self, model_name: str, scaler_variant: str, version: int | None = None) -> bool:
```
- Deletes a specific version or all versions of a model
- Returns True if something was deleted

```python
def save_winner(
    self,
    winner_result: WinnerResult,
    pipeline: Pipeline,
    feature_names: list[str],
) -> str:
```
- Convenience method: calls `save_model` with `is_winner=True` and rank from WinnerResult
