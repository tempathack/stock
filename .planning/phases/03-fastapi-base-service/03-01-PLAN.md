- `router = APIRouter()`

**main.py** — Replace stub with FastAPI app:
- Import `AsyncIterator` from `collections.abc`, `asynccontextmanager` from `contextlib`, `FastAPI` from `fastapi`
- Import `settings` from `app.config`, `health` from `app.routers`, `get_logger` from `app.utils.logging`
- `logger = get_logger(__name__)`
- `@asynccontextmanager async def lifespan(app: FastAPI) -> AsyncIterator[None]:` — logs "service starting" with service, version, environment fields on startup; logs "service shutting down" on shutdown
- `app = FastAPI(title="Stock Prediction API", version=settings.APP_VERSION, lifespan=lifespan)`
- `app.include_router(health.router)` — ONLY health router; do NOT include ingest, predict, models, or market routers

Do NOT use `on_startup`/`on_shutdown` (deprecated in FastAPI 0.111.0). Use lifespan context manager per user decision.
Do NOT import or use `os.getenv()` in any router — use `settings` singleton only.
