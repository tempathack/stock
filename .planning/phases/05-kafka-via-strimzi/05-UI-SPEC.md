## Applicability

**This phase has NO user interface.** Phase 5 (Kafka via Strimzi) is a pure Kubernetes infrastructure phase. It deploys the Strimzi operator, a KRaft-mode Kafka broker, and two KafkaTopic CRs into the `storage` namespace. All artifacts are YAML manifests and shell scripts. There are no visual components, web pages, or user-facing interactions.

The first frontend phase is Phase 25 (React App Bootstrap). UI design contracts become relevant starting there.
