## Design Decisions
- **Mock at external boundaries** — Yahoo Finance HTTP and Kafka broker are mocked; everything between is real code
- **Cross-service imports** — conftest adds both service directories to sys.path so tests can import from both API and consumer
- **No live infrastructure required** — Tests run in any Python environment with dependencies installed
- **Deterministic fixtures** — All test data is hardcoded, no randomness
- **Separate integration directory** — Keeps integration tests distinct from unit tests; can be run selectively via `pytest tests/integration/`
