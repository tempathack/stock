## Design Decisions
- **Synthetic data, not DB** — Uses `data_dict` parameter to bypass database dependency; tests pipeline logic in isolation
- **skip_shap=True** — SHAP has numba/numpy 2.x compatibility issues; shap explainability tested separately in Phase 16
- **tmp_path for isolation** — Each test run uses fresh temporary directories; no state leakage between runs
- **@pytest.mark.slow** — Allows CI to skip expensive tests with `-m "not slow"` while still validating in nightly runs
- **Round-trip validation** — Tests cover the full cycle: train → persist → deploy → load → predict → save → verify
