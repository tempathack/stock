## Steps

### 1. Add `train_tree_models()` to `model_trainer.py`

- Same pattern as `train_linear_models()`
- Iterates TREE_MODELS + BOOSTER_MODELS × SCALER_VARIANTS
- Returns list[TrainingResult] sorted by OOS RMSE
- Up to 27 runs (9 models × 3 scalers) if all boosters available

### 2. Add `train_all_models()` convenience function

- Calls `train_linear_models()` + `train_tree_models()`, merges and sorts results
- Convenience for downstream phases

### 3. Extend `ml/tests/test_model_trainer.py`

- TestTrainTreeModels: returns correct count (18 sklearn + up to 9 booster)
- Test sorted by RMSE
- Test all model names present
- Test all scaler variants present
- Test single tree model (RF with tuning)
- Test single booster model (XGBoost)
