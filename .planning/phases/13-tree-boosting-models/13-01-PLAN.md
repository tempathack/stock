## Steps

### 1. Add `TREE_MODELS` dict to `model_configs.py`

Six sklearn entries with search spaces:
- `random_forest`: RandomForestRegressor — n_estimators, max_depth, min_samples_split, max_features; n_iter=50
- `gradient_boosting`: GradientBoostingRegressor — n_estimators, learning_rate, max_depth, subsample; n_iter=50
- `hist_gradient_boosting`: HistGradientBoostingRegressor — learning_rate, max_iter, max_depth; n_iter=30
- `extra_trees`: ExtraTreesRegressor — n_estimators, max_depth; n_iter=30
- `decision_tree`: DecisionTreeRegressor — max_depth, min_samples_split; search_space provided
- `adaboost`: AdaBoostRegressor — n_estimators, learning_rate; n_iter=30

All with `random_state=42` in default_params.

### 2. Add `BOOSTER_MODELS` dict (conditional imports)

- XGBRegressor: try import, n_estimators/learning_rate/max_depth/subsample/colsample_bytree; verbosity=0, n_iter=50
- LGBMRegressor: try import, n_estimators/learning_rate/num_leaves/max_depth; verbose=-1, n_iter=50
- CatBoostRegressor: try import, iterations/learning_rate/depth; verbose=0, n_iter=30

### 3. Register both families at module level

