## Interaction Contract

### Loading State

- Gate DataGrid rendering on `results.every(r => !r.isLoading)` (all 4 horizon queries complete).
- While loading: render `LoadingSpinner` centered in the table area тАФ do NOT show partial data (prevents layout shift from column count changes).
- Loading indicator placement: replace the DataGrid entirely with a centered `Box` containing `LoadingSpinner` + caption text "Loading forecasts for all horizonsтАж".

### Error State

- If ALL 4 queries fail: show `ErrorFallback` component (already exists at `components/ui/ErrorFallback.tsx`).
- If PARTIAL failure (1-3 horizons fail): render table with available horizon columns only. Missing horizon columns display `тАФ` in every cell. Add a MUI `Alert severity="warning"` above the table with the partial error message (see Copywriting Contract below).

### Row Selection

- Single click on any row тЖТ `onSelectTicker(row.ticker)` тЖТ opens stock detail drawer (existing behavior, unchanged).
- Selected row: `Mui-selected` CSS class applied via `getRowClassName` (existing pattern, unchanged).
- `getRowId`: `(row: MultiHorizonForecastRow) => row.ticker` тАФ one row per ticker, no collision.

### Sorting

- Default sort: `return_7d` descending (highest 7-day expected return at top).
- All return % columns are sortable.
- Price and daily change columns are sortable.
- Company and sector columns are sortable.

### Return % Cell Coloring

- `value >= 0`: `color: "success.main"` (`#00FF87`), prefix `+`.
- `value < 0`: `color: "error.main"` (`#FF2D78`), no prefix.
- `value == null`: `тАФ` in `text.secondary` (`#6B60A8`), no color.

### Predicted Price Cell

- Format: `$NNN.NN` (monospace, 2 decimal places).
- Tooltip on hover: show `Predicted date: {predicted_date}` using MUI `Tooltip` (`sx` from `MuiTooltip` override in theme тАФ dark glass, JetBrains Mono 0.7rem).
- Note: KServe returns same price value across all horizons (horizon only affects `predicted_date`). Label columns with predicted date in tooltip, not just a standalone number.

### Pagination

- Default page size: 25 rows.
- Page size options: [25, 50, 100] (matches existing `ForecastTable` тАФ unchanged).
