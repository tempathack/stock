<interfaces>
Response schema:
```python
class MacroLatestResponse(BaseModel):
    as_of_date: date | None
    vix: float | None           # from feast_yfinance_macro (^VIX close)
    spy_return: float | None    # from feast_yfinance_macro
    dgs10: float | None         # 10Y Treasury yield
    t10y2y: float | None        # 10Y-2Y spread (yield curve)
    baml_hy_oas: float | None   # HY credit spread
    wti_crude: float | None     # WTI oil price
    usd_broad: float | None     # Broad USD index
    icsa: float | None          # Initial jobless claims (thousands)
    core_pce: float | None      # Core PCE YoY %
    dgs2: float | None          # 2Y Treasury yield
    t10yie: float | None        # 10Y breakeven inflation
```

SQL to fetch:
```sql
-- FRED indicators
SELECT date AS as_of_date, dgs10, t10y2y, baml_hy_oas, wti_crude, usd_broad, icsa, core_pce, dgs2, t10yie
FROM macro_fred_daily ORDER BY date DESC LIMIT 1;

-- yfinance macro (market-wide, use any ticker e.g. SPY itself)
SELECT timestamp::date AS as_of_date, vix, spy_return
FROM feast_yfinance_macro WHERE ticker = 'SPY' ORDER BY timestamp DESC LIMIT 1;
```
