prediction horizon. The toggle triggers a React Query refetch with the new horizon parameter,
and the page updates to show predictions for the selected horizon. A new useAvailableHorizons
hook discovers which horizons are deployed.

horizon immediately loads the corresponding predictions. The default is 7D, matching the
pre-Phase 43 behavior. The toggle is accessible (ARIA radiogroup) and responsive.
