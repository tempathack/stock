# Plan 11-02: Scaler Pipeline Factory

**Phase:** 11 — Lag Features & Transformer Pipelines
**Scope:** FEAT-17, FEAT-18, FEAT-19
**File:** `ml/features/transformations.py`
**Tests:** `ml/tests/test_transformations.py`

---

## Steps

### 1. Implement `build_scaler_pipeline(variant)`
- Accepts variant string: `"standard"`, `"quantile"`, `"minmax"`
- Returns `sklearn.pipeline.Pipeline` with single scaler step
- Mappings:
  - `"standard"` → `StandardScaler()`
  - `"quantile"` → `QuantileTransformer(output_distribution="normal", random_state=42)`
  - `"minmax"` → `MinMaxScaler()`
- Raises `ValueError` for unknown variants
- Export `SCALER_VARIANTS = ("standard", "quantile", "minmax")` constant

### 2. Write tests in `ml/tests/test_transformations.py`
- TestBuildScalerPipeline: returns Pipeline, correct scaler type per variant, unknown variant raises ValueError
- TestStandardScaler: fit_transform produces ~zero mean / ~unit std
- TestQuantileTransformer: fit_transform produces approximately normal distribution
- TestMinMaxScaler: fit_transform values in [0, 1]
- TestNoLeakage: fit on train, transform on test — train stats don't leak into test set

### 3. Update `ml/features/__init__.py`
- Export `build_scaler_pipeline` and `SCALER_VARIANTS`
