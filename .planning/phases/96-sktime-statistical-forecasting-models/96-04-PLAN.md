# Plan 96-04: sktime Regression Model Wrappers

## Goal
Add sklearn-compatible wrappers for sktime's `regression` module models — MiniROCKET,
ROCKET, TimeSeriesForest, RandomInterval, and Catch22 — that reshape the existing 2D
tabular feature matrix into the 3D `(n_samples, 1, n_timepoints)` format sktime regressors
expect, so they plug into `_build_pipeline()` and `train_single_model()` without changes.

---

## Context

### Why sktime Regressors Are Different from Forecasting Wrappers (Plan 96-01)

The forecasting wrappers in `sktime_wrappers.py` use a complex fit-on-y approach because
sktime *forecasters* predict future values of the series. sktime *regressors* are standard
supervised learners: `fit(X, y)` / `predict(X)` — semantically identical to sklearn.

The only integration challenge is the input format:
- **Existing pipeline**: `X` is 2D `(n_samples, n_lag_features)` — tabular lag/indicator matrix
- **sktime regressors**: `X` must be 3D `(n_instances, n_dimensions, series_length)`

**Reshape strategy:** `X.reshape(n, 1, T)` — treat each sample's feature vector as a
univariate time series of length T (one "channel"). This is semantically valid because:
- The feature vector already contains ordered lag features (lag_1, lag_2, ... lag_N)
- ROCKET's random convolutional kernels detect local patterns across these lags
- TimeSeriesForest extracts interval statistics over the lag window

This is equivalent to applying ROCKET directly to the raw lag-feature window.

### CPU-Eligible Regression Models

From sktime.regression, the following are CPU-trainable and dependency-light:

| Model | Class | Extra Dep | Speed | Notes |
|---|---|---|---|---|
| MiniROCKET | `RocketRegressor(rocket_transform="minirocket")` | `numba` | Very fast | State-of-the-art accuracy |
| ROCKET | `RocketRegressor(rocket_transform="rocket")` | `numba` | Fast | Original paper model |
| TimeSeriesForest | `TimeSeriesForestRegressor` | none | Moderate | No extra deps; interpretable |
| RandomInterval | `RandomIntervalRegressor` | none | Moderate | Interval ensemble |
| Catch22 | `Catch22Regressor` | `pycatch22` | Fast | 22 canonical TS features |

**Not included:**
- Deep learning models (CNN, LSTM, ResNet, InceptionTime) — require TensorFlow; GPU-focused
- `KNeighborsTimeSeriesRegressor` — O(n²) predict time; too slow for hundreds of stocks
- `FreshPRINCERegressor` / `TSFreshRegressor` — tsfresh is very slow (thousands of features)
- `TimeSeriesSVRTslearn` — quadratic scaling; not production-grade at our data volume

```python
# ===========================================================================
# sktime Regression Module Wrappers

class SktimeRegressionWrapper(BaseEstimator, RegressorMixin):
    """Thin adapter that reshapes 2D tabular X → 3D for sktime regressors.

    sktime regressors expect X of shape (n_instances, n_dimensions, series_length).
    The existing pipeline produces X of shape (n_samples, n_features).

    This wrapper reshapes X to (n_samples, 1, n_features) — treating each sample's
    feature vector as a univariate time series of length n_features — before passing
    it to the underlying sktime regressor. The sktime model's fit/predict output is
    passed through unchanged.

    Parameters
    ----------
    regressor : sktime BaseRegressor instance
        Any sktime regression estimator supporting fit(X, y) / predict(X) with
        numpy3D input format.
    """

    def __init__(self, regressor):
        self.regressor = regressor

    def fit(self, X: np.ndarray, y: np.ndarray):
        from sklearn.base import clone

        X_3d = X.reshape(X.shape[0], 1, X.shape[1])  # (n, 1, T)
        self.regressor_ = clone(self.regressor)
        self.regressor_.fit(X_3d, y)
        return self

    def predict(self, X: np.ndarray) -> np.ndarray:
        check_is_fitted(self, "regressor_")
        X_3d = X.reshape(X.shape[0], 1, X.shape[1])
        return self.regressor_.predict(X_3d)


class MiniRocketWrapper(SktimeRegressionWrapper):
    """MiniROCKET — fastest ROCKET variant, state-of-the-art accuracy on CPU.

    Uses random convolutional kernels (MiniRocket transform) followed by a
    Ridge classifier. ~75x faster than original ROCKET on large datasets.
    Requires: numba (JIT compilation; results cached after first run).
    """

    def __init__(self, num_kernels: int = 10_000, max_dilations_per_kernel: int = 32):
        from sktime.regression.kernel_based import RocketRegressor

        super().__init__(
            regressor=RocketRegressor(
                rocket_transform="minirocket",
                num_kernels=num_kernels,
                max_dilations_per_kernel=max_dilations_per_kernel,
            )
        )
        self.num_kernels = num_kernels
        self.max_dilations_per_kernel = max_dilations_per_kernel


class RocketWrapper(SktimeRegressionWrapper):
    """ROCKET — original random convolutional kernel transform regression.

    Requires: numba.
    """

    def __init__(self, num_kernels: int = 10_000):
        from sktime.regression.kernel_based import RocketRegressor

        super().__init__(
            regressor=RocketRegressor(
                rocket_transform="rocket",
                num_kernels=num_kernels,
            )
        )
        self.num_kernels = num_kernels


class TimeSeriesForestWrapper(SktimeRegressionWrapper):
    """TimeSeriesForestRegressor — interval-based ensemble, no extra deps.

    Uses mean, std, and slope summary statistics over random time series
    intervals. Interpretable and dependency-free.
    """

    def __init__(self, n_estimators: int = 200, min_interval: int = 3,
                 n_jobs: int = -1):
        from sktime.regression.interval_based import TimeSeriesForestRegressor

        super().__init__(
            regressor=TimeSeriesForestRegressor(
                n_estimators=n_estimators,
                min_interval=min_interval,
                n_jobs=n_jobs,
            )
        )
        self.n_estimators = n_estimators
        self.min_interval = min_interval
        self.n_jobs = n_jobs


class RandomIntervalWrapper(SktimeRegressionWrapper):
    """RandomIntervalRegressor — random interval features + sklearn regressor.

    Extracts multiple summary statistics from random intervals, then applies
    a sklearn regressor (default: Ridge). No extra deps beyond core sktime.
    """

    def __init__(self, n_estimators: int = 200, n_jobs: int = -1):
        from sktime.regression.interval_based import RandomIntervalRegressor

        super().__init__(
            regressor=RandomIntervalRegressor(
                n_estimators=n_estimators,
                n_jobs=n_jobs,
            )
        )
        self.n_estimators = n_estimators
        self.n_jobs = n_jobs


class Catch22Wrapper(SktimeRegressionWrapper):
    """Catch22Regressor — 22 canonical time series features + sklearn regressor.

    Extracts 22 interpretable statistical features (Catch22) then applies a
    linear classifier. Fast inference, lightweight.
    Requires: pycatch22.
    """

    def __init__(self, outlier_norm: bool = False, replace_nans: bool = True):
        from sktime.regression.feature_based import Catch22Regressor

        super().__init__(
            regressor=Catch22Regressor(
                outlier_norm=outlier_norm,
                replace_nans=replace_nans,
            )
        )
        self.outlier_norm = outlier_norm
        self.replace_nans = replace_nans
```

### 3. Verify import check for regression wrappers

```bash
python -c "
import numpy as np
from ml.models.sktime_wrappers import (
    MiniRocketWrapper, RocketWrapper, TimeSeriesForestWrapper,
    RandomIntervalWrapper, Catch22Wrapper
)

np.random.seed(42)
X = np.random.randn(300, 40)   # 300 samples, 40 lag features
y = np.cumsum(np.random.randn(300)) * 0.01

for cls in [TimeSeriesForestWrapper, RandomIntervalWrapper, Catch22Wrapper]:
    m = cls()
    m.fit(X[:240], y[:240])
    preds = m.predict(X[240:])
    assert len(preds) == 60
    print(f'{cls.__name__}: OK')

# ROCKET requires numba JIT on first run - may take 30-60s to compile
for cls in [MiniRocketWrapper, RocketWrapper]:
    m = cls(num_kernels=1000)   # reduced for smoke test
    m.fit(X[:240], y[:240])
    preds = m.predict(X[240:])
    assert len(preds) == 60
    print(f'{cls.__name__}: OK (numba compiled)')

print('All regression wrappers OK')
"
```
