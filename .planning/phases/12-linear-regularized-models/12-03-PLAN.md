# Plan 12-03: Model Trainer — Train All 6 Linear Models × 3 Scalers

**Phase:** 12 — Linear & Regularized Models
**Scope:** MODEL-01–MODEL-06, MODEL-19, MODEL-20
**Files:** `ml/pipelines/components/model_trainer.py`
**Tests:** `ml/tests/test_model_trainer.py`

---

## Steps

### 1. Implement `train_single_model(config, X_train, y_train, X_test, y_test, scaler_variant, cv) -> TrainingResult`

- Build pipeline: `Pipeline([("scaler", build_scaler_pipeline(variant).steps[0]), ("model", config.model_class(**config.default_params))])`
- If `config.search_space` is not None:
  - Prefix search space keys with `model__` for pipeline
  - Run `RandomizedSearchCV(pipeline, param_distributions, cv=cv, n_iter=config.n_iter, scoring="neg_root_mean_squared_error", random_state=42)`
  - Refit best estimator
- Else: fit pipeline directly
- Compute per-fold metrics via walk_forward_evaluate
- Compute OOS metrics on test set
- Return TrainingResult

### 2. Implement `train_linear_models(X_train, y_train, X_test, y_test) -> list[TrainingResult]`

- Iterates over all LINEAR_MODELS × SCALER_VARIANTS (6×3 = 18 runs)
- Calls `train_single_model` for each
- Returns list of 18 TrainingResults

### 3. Implement `save_training_results(results: list[TrainingResult], output_dir: str) -> None`

- Saves JSON summary (metrics, params) to `{output_dir}/training_results.json`
- Pickles each pipeline to `{output_dir}/pipelines/{model_name}_{scaler}.pkl`

### 4. Write `ml/tests/test_model_trainer.py`

- Uses small synthetic dataset (conftest fixture + features + target)
- Tests `train_single_model` with LinearRegression (no tuning)
- Tests `train_single_model` with Ridge (with tuning)
- Tests `train_linear_models` returns 18 results
- Tests `save_training_results` writes JSON + pickle files
- Tests pipeline predict works after reload
