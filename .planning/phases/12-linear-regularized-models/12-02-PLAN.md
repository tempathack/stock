## Steps

### 1. Define `TrainingResult` dataclass in `model_configs.py`

```python
@dataclass
class TrainingResult:
    model_name: str
    scaler_variant: str
    best_params: dict
    fold_metrics: list[dict]   # per-fold metrics
    oos_metrics: dict           # aggregated OOS metrics
    fold_stability: float       # std of OOS RMSE across folds
    pipeline_path: str | None   # path to pickled pipeline (set after save)
```

### 2. Define `ModelConfig` dataclass

```python
@dataclass
class ModelConfig:
    name: str
    model_class: type
    default_params: dict
    search_space: dict | None   # None = no tuning
    n_iter: int = 30
```

### 3. Define `LINEAR_MODELS` dict

Six entries:
- `linear_regression`: LinearRegression, no search space
- `ridge`: Ridge, alpha search space
- `lasso`: Lasso, alpha search space
- `elastic_net`: ElasticNet, alpha + l1_ratio
- `bayesian_ridge`: BayesianRidge, no search space
- `huber`: HuberRegressor, epsilon search space

### 4. Helper functions
- `get_model_configs(family="linear") -> dict[str, ModelConfig]`
- `get_all_model_configs() -> dict[str, ModelConfig]` (for multi-phase use)

### 5. Update `ml/models/__init__.py` with exports

### 6. Write `ml/tests/test_model_configs.py`
- All 6 linear models present
- Each ModelConfig has valid sklearn class
- Search spaces have expected keys
- TrainingResult serializable
