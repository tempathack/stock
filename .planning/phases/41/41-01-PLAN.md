<objective>
Create a K8s CronJob that runs pg_dump daily at 04:00 UTC to back up the stockdb database,
storing custom-format dump files on a dedicated PVC with automated retention (7 daily + 4 weekly).
Wire the new manifests into deploy-all.sh.

Purpose: DBHARD-08 (Daily pg_dump backup CronJob with 7-day retention).
Output: CronJob runs nightly producing /backups/stockdb_YYYYMMDD_HHMMSS.dump files.
Retention script prunes old backups: keeps 7 most recent days + Sundays for 4 weeks.
Manual restore: `kubectl exec -n storage <pg-pod> -- pg_restore -U stockuser -d stockdb /backups/<file>.dump`
</objective>
