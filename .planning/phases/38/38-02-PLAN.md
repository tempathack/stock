  <name>Task 1: Create k8s/monitoring/grafana-dashboard-api-health.yaml</name>
  <files>k8s/monitoring/grafana-dashboard-api-health.yaml</files>
  <action>

Create a ConfigMap wrapping a Grafana dashboard JSON with the following panels:

**Row 1 — HTTP Overview:**
- Panel: "Request Rate" — `sum(rate(http_requests_total{kubernetes_namespace="$namespace"}[5m]))` — timeseries
- Panel: "Error Rate %" — `100 * sum(rate(http_requests_total{kubernetes_namespace="$namespace",status=~"5.."}[5m])) / sum(rate(http_requests_total{kubernetes_namespace="$namespace"}[5m]))` — stat with thresholds (green < 1, yellow < 5, red >= 5)

**Row 2 — Latency:**
- Panel: "p50 Latency" — `histogram_quantile(0.50, sum(rate(http_request_duration_seconds_bucket{kubernetes_namespace="$namespace"}[5m])) by (le))` — stat, unit: seconds
- Panel: "p95 Latency" — same with 0.95 quantile — stat
- Panel: "p99 Latency" — same with 0.99 quantile — stat
- Panel: "Latency Distribution" — all three quantiles as timeseries

**Row 3 — Predictions:**
- Panel: "Prediction Request Rate" — `sum(rate(prediction_requests_total{kubernetes_namespace="$namespace"}[5m])) by (status)` — timeseries
- Panel: "Prediction Latency p95" — `histogram_quantile(0.95, sum(rate(prediction_latency_seconds_bucket{kubernetes_namespace="$namespace"}[5m])) by (le))` — stat
- Panel: "Inference Errors" — `sum(rate(model_inference_errors_total{kubernetes_namespace="$namespace"}[5m])) by (error_type)` — timeseries

Template variable: `namespace` with query `label_values(http_requests_total, kubernetes_namespace)`, default: `ingestion`.
