The deployer copies the winner model artifacts to a serving directory, writes a `serving_config.json`, and marks the model as active. Full test coverage. ## Design Decisions

1. **Registry `is_active` tracking:** Three new methods on `ModelRegistry`:
   - `activate_model(model_name, scaler_variant, version)` — reads `metadata.json`, sets `is_active=True`, writes back. Raises `FileNotFoundError` if model/version not found.
   - `deactivate_all()` — iterates all model versions, sets `is_active=False` in each `metadata.json`. Returns count of deactivated models.
   - `get_active_model()` — scans all metadata files, returns the one with `is_active=True`, or `None`. If multiple active (shouldn't happen), returns the first found.

2. **`deploy_winner_model()` function:** Single entry point that orchestrates the full deployment flow. Uses `shutil.copytree` / `shutil.copy2` to copy artifacts, ensuring the serving directory is self-contained.

3. **`serving_config.json`  schema:**
   ```json
   {
     "model_name": "ridge",
     "scaler_variant": "standard",
     "version": 1,
     "features": ["close", "rsi_14", ...],
     "artifact_path": "/models/active",
     "deployed_at": "2026-03-20T12:00:00+00:00",
     "is_active": true,
     "oos_metrics": { ... }
   }
   ```

4. **Idempotent deployment:** If the serving directory already exists, it is cleared (via `shutil.rmtree`) before copying new artifacts. This ensures clean deployments with no stale files.

5. **Artifact copying:** Only the version directory contents are copied (pipeline.pkl, metadata.json, features.json, and any SHAP files). The nested registry directory structure is flattened to the serving path.

6. **Test isolation:** All tests use `tmp_path` fixtures. A `populated_registry` fixture pre-saves a winner + runners-up using `select_and_persist_winner()`, providing a realistic registry state for deployment tests.

      ```python
      # ------------------------------------------------------------------
      # Activation
      # ------------------------------------------------------------------

      def activate_model(
          self,
          model_name: str,
          scaler_variant: str,
          version: int,
      ) -> None:
          """Mark a specific model version as active for serving.

          Raises ``FileNotFoundError`` if the model or version does not exist.
          """
          model_key = f"{model_name}_{scaler_variant}"
          ver_dir = self._base / model_key / f"v{version}"
          meta_path = ver_dir / "metadata.json"

          if not meta_path.exists():
              raise FileNotFoundError(
                  f"Model {model_key} v{version} not found in registry."
              )

          with open(meta_path) as f:
              metadata = json.load(f)

          metadata["is_active"] = True

          with open(meta_path, "w") as f:
              json.dump(metadata, f, indent=2, default=str)

          logger.info("Activated %s v%d.", model_key, version)

      def deactivate_all(self) -> int:
          """Set is_active=False for all model versions. Returns count deactivated."""
          count = 0
          if not self._base.exists():
              return count

          for model_dir in self._base.iterdir():
              if not model_dir.is_dir():
                  continue
              for ver_dir in model_dir.iterdir():
                  meta_path = ver_dir / "metadata.json"
                  if not meta_path.exists():
                      continue
                  with open(meta_path) as f:
                      metadata = json.load(f)
                  if metadata.get("is_active", False):
                      metadata["is_active"] = False
                      with open(meta_path, "w") as f:
                          json.dump(metadata, f, indent=2, default=str)
                      count += 1

          logger.info("Deactivated %d model version(s).", count)
          return count

      def get_active_model(self) -> dict | None:
          """Return metadata for the currently active model, or None."""
          if not self._base.exists():
              return None

          for model_dir in sorted(self._base.iterdir()):
              if not model_dir.is_dir():
                  continue
              for ver_dir in sorted(model_dir.iterdir()):
                  meta_path = ver_dir / "metadata.json"
                  if not meta_path.exists():
                      continue
                  with open(meta_path) as f:
                      metadata = json.load(f)
                  if metadata.get("is_active", False):
                      return {
                          "model_name": metadata.get("model_name"),
                          "scaler_variant": metadata.get("scaler_variant"),
                          "version": metadata.get("version"),
                          "is_active": True,
                          "is_winner": metadata.get("is_winner", False),
                          "oos_rmse": metadata.get("oos_metrics", {}).get("rmse"),
                          "path": str(ver_dir),
                      }

          return None
      ```

      ```python
      def deploy_winner_model(
          registry_dir: str = "model_registry",
          serving_dir: str = "/models/active",
      ) -> dict:
          """Deploy the winner model from the registry to the serving directory.

          Steps:
          1. Find the current winner in the registry
          2. Deactivate any previously active models
          3. Clear and recreate the serving directory
          4. Copy winner artifacts (pipeline.pkl, metadata.json, features.json, SHAP files)
          5. Write serving_config.json
          6. Activate the winner in the registry
          7. Return deployment summary

          Raises ``FileNotFoundError`` if no winner exists in the registry.
          """
      ```

      Implementation:
      - Create `ModelRegistry(base_dir=registry_dir)`
      - Call `registry.get_winner()` — raise `FileNotFoundError` if `None`
      - Parse model_name, scaler_variant from winner metadata
      - Call `registry.load_model(model_name, scaler_variant, version)` to get full artifact info
      - Call `registry.deactivate_all()`
      - Create serving path: `Path(serving_dir)`, clear if exists (`shutil.rmtree`), then `mkdir(parents=True)`
      - Copy all files from winner version directory to serving_dir using `shutil.copy2`
      - Build `serving_config` dict with model_name, scaler_variant, version, features (from features.json),
        artifact_path (str of serving_dir), deployed_at (UTC ISO), is_active=True, oos_metrics
      - Write `serving_config.json` to serving_dir
      - Call `registry.activate_model(model_name, scaler_variant, version)`
      - Log deployment summary
      - Return dict: model_name, scaler_variant, version, serving_path, deployed_at
