# Plan 19-02 — End-to-End Pipeline Integration Verification

## Objective
Verify that the full KF-09 → KF-12 pipeline flow (explainability → winner selection → model persistence → deployment) works end-to-end as an integrated sequence. This plan adds an integration test that chains all four Phase 19 components in the order they will execute within the Kubeflow pipeline. No new production code — tests only. Validates **KF-09, KF-10, KF-11, KF-12** as a cohesive unit.

## Dependencies
- `ml/pipelines/components/explainer.py` — `explain_top_models()` (Phase 16)
- `ml/pipelines/components/model_selector.py` — `select_and_persist_winner()` (Phase 15)
- `ml/pipelines/components/deployer.py` — `deploy_winner_model()` (Plan 19-01)
- `ml/models/registry.py` — `ModelRegistry` with activation methods (Plan 19-01)
- `ml/evaluation/ranking.py` — `rank_models()`, `RankedModel` (Phase 15)
- `ml/tests/conftest.py` — `sample_ohlcv_df` fixture

## Design Decisions

1. **Integration test, not unit test:** This test chains real calls to `select_and_persist_winner()` → `explain_top_models()` → `deploy_winner_model()`, using synthetic `TrainingResult` objects and fitted pipelines. No mocking of the component functions themselves.

2. **Synthetic models for speed:** Uses fitted Ridge, Lasso, RandomForest, KNN, SVR pipelines (same pattern as `test_explainer.py`). Avoids the full training pipeline to keep runtime under 30 seconds.

3. **Verification checkpoints:** After each pipeline step, the test asserts intermediate state:
   - After `select_and_persist_winner`: winner saved, registry has 5 models
   - After `explain_top_models`: SHAP files exist in registry
   - After `deploy_winner_model`: serving directory populated, model active, serving_config valid

4. **Round-trip validation:** The final assertion loads the deployed pipeline from the serving directory and calls `.predict()` to verify it produces valid predictions.

      ```python
      def test_full_pipeline_flow(self, pipeline_artifacts):
          """Chain: select_winner → explain → deploy → verify serving."""
          arts = pipeline_artifacts
          
          # Step 1: KF-10/KF-11 — Winner selection + persistence
          winner_info = select_and_persist_winner(
              arts["results"], arts["pipelines"], arts["features"],
              registry_dir=arts["registry_dir"],
          )
          assert winner_info["total_candidates"] == 5
          registry = ModelRegistry(base_dir=arts["registry_dir"])
          assert registry.get_winner() is not None
          assert len(registry.list_models()) == 5

          # Step 2: KF-09 — Explainability
          shap_output = explain_top_models(
              arts["results"], arts["pipelines"], arts["X"], arts["features"],
              registry_dir=arts["registry_dir"], top_n=5,
          )
          assert len(shap_output) >= 1
          # Verify SHAP files in registry
          shap_files = list(Path(arts["registry_dir"]).rglob("shap_importance.json"))
          assert len(shap_files) >= 1

          # Step 3: KF-12 — Deployment
          deploy_info = deploy_winner_model(
              registry_dir=arts["registry_dir"],
              serving_dir=arts["serving_dir"],
          )
          assert "model_name" in deploy_info
          assert "deployed_at" in deploy_info

          # Verify serving directory
          serving = Path(arts["serving_dir"])
          assert (serving / "serving_config.json").exists()
          assert (serving / "pipeline.pkl").exists()
          assert (serving / "metadata.json").exists()
          assert (serving / "features.json").exists()

          # Verify serving config
          with open(serving / "serving_config.json") as f:
              config = json.load(f)
          assert config["is_active"] is True
          assert config["model_name"] == deploy_info["model_name"]

          # Verify model is active in registry
          active = registry.get_active_model()
          assert active is not None
          assert active["is_active"] is True

          # Round-trip: load deployed pipeline and predict
          with open(serving / "pipeline.pkl", "rb") as f:
              loaded_pipeline = pickle.load(f)
          preds = loaded_pipeline.predict(arts["X"][:5])
          assert preds.shape == (5,)
          assert not np.any(np.isnan(preds))
      ```
