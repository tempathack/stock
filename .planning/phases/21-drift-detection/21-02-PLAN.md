# Plan 21-02 — Drift Monitor + Drift Logger + Trigger Bridge + Tests

## Objective
Implement the drift check orchestrator (`monitor.py`), drift event logger (`trigger.py`), and bridge to the retraining pipeline. Satisfies **DRIFT-04** (daily drift check job) and **DRIFT-05** (drift events logged to drift_logs table).

## Dependencies
- `ml/drift/detector.py` — DataDriftDetector, PredictionDriftDetector, ConceptDriftDetector, DriftResult (Plan 21-01)
- `ml/pipelines/drift_pipeline.py` — `trigger_retraining()` (Phase 20)
- `db/init.sql` — drift_logs table schema (Phase 4)

## Design Decisions

1. **`DriftMonitor` class in monitor.py:** Orchestrates all three drift detectors:
   ```python
   @dataclass
   class DriftCheckResult:
       checked_at: str
       data_drift: DriftResult
       prediction_drift: DriftResult
       concept_drift: DriftResult
       any_drift: bool  # True if any detector flagged drift

       def to_dict(self) -> dict:
           return asdict(self)

   class DriftMonitor:
       def __init__(
           self,
           ks_threshold: float = 0.01,
           psi_threshold: float = 0.2,
           error_multiplier: float = 1.5,
           degradation_threshold: float = 1.3,
       ):
       def check(
           self,
           reference_features: pd.DataFrame,
           current_features: pd.DataFrame,
           baseline_errors: np.ndarray,
           recent_errors: np.ndarray,
           historical_rmse: float,
           recent_rmse: float,
       ) -> DriftCheckResult:
   ```

2. **`DriftLogger` class in trigger.py:** Persists drift events via multiple backends:
   ```python
   class DriftLogger:
       def __init__(self, log_dir: str = "drift_logs", db_settings: DBSettings | None = None):
       def log_event(self, result: DriftResult) -> None:
       def log_check(self, check_result: DriftCheckResult) -> None:
       def _log_to_file(self, result: DriftResult) -> None:
       def _log_to_db(self, result: DriftResult) -> None:
       def get_recent_events(self, n: int = 50) -> list[dict]:
   ```
   - File backend: appends JSON to `{log_dir}/drift_events.jsonl` (always available)
   - DB backend: inserts into drift_logs table (optional, gated on psycopg2)

3. **`evaluate_and_trigger()` function in trigger.py:**
   ```python
   def evaluate_and_trigger(
       reference_features: pd.DataFrame,
       current_features: pd.DataFrame,
       baseline_errors: np.ndarray,
       recent_errors: np.ndarray,
       historical_rmse: float,
       recent_rmse: float,
       tickers: list[str] | None = None,
       registry_dir: str = "model_registry",
       serving_dir: str = "/models/active",
       log_dir: str = "drift_logs",
       auto_retrain: bool = True,
       skip_shap: bool = False,
   ) -> DriftCheckResult:
   ```
   - Runs DriftMonitor.check()
   - Logs all drift events via DriftLogger
   - If any_drift and auto_retrain: calls trigger_retraining() with appropriate reason
   - Returns DriftCheckResult

4. **Update `ml/drift/__init__.py`:** Export all public classes and functions.
