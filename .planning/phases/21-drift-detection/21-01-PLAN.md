## Design Decisions

1. **`DriftResult` dataclass:** Common return type for all three detectors:
   ```python
   @dataclass
   class DriftResult:
       drift_type: str           # "data_drift" | "prediction_drift" | "concept_drift"
       is_drifted: bool
       severity: str             # "none" | "low" | "medium" | "high"
       details: dict
       timestamp: str            # ISO timestamp
       features_affected: list[str] = field(default_factory=list)

       def to_dict(self) -> dict:
           return asdict(self)
   ```

2. **`DataDriftDetector` class:** Stateless detector, instantiated with thresholds:
   ```python
   class DataDriftDetector:
       def __init__(self, ks_threshold: float = 0.01, psi_threshold: float = 0.2, n_bins: int = 10):
       def detect(self, reference: pd.DataFrame, current: pd.DataFrame) -> DriftResult:
       @staticmethod
       def _compute_psi(reference: np.ndarray, current: np.ndarray, n_bins: int) -> float:
       @staticmethod
       def _ks_test(reference: np.ndarray, current: np.ndarray) -> tuple[float, float]:
   ```

3. **`PredictionDriftDetector` class:**
   ```python
   class PredictionDriftDetector:
       def __init__(self, error_multiplier: float = 1.5):
       def detect(self, baseline_errors: np.ndarray, recent_errors: np.ndarray) -> DriftResult:
   ```
   - `baseline_errors`: historical absolute prediction errors
   - `recent_errors`: recent absolute prediction errors
   - Drift if `mean(recent_errors) > error_multiplier * mean(baseline_errors)`

4. **`ConceptDriftDetector` class:**
   ```python
   class ConceptDriftDetector:
       def __init__(self, degradation_threshold: float = 1.3):
       def detect(self, historical_rmse: float, recent_rmse: float) -> DriftResult:
   ```
   - Drift if `recent_rmse > degradation_threshold * historical_rmse`

5. **Severity mapping:**
   - `ratio < 1.0` → "none"
   - `1.0 ≤ ratio < threshold` → "none" (within acceptable range)
   - `threshold ≤ ratio < 2×threshold` → "medium"
   - `ratio ≥ 2×threshold` → "high"
   For DataDriftDetector, severity based on fraction of features drifted.

6. **PSI implementation:** Hand-rolled, no external library:
   ```python
   @staticmethod
   def _compute_psi(reference: np.ndarray, current: np.ndarray, n_bins: int = 10) -> float:
       # Quantile-based binning from reference distribution
       breakpoints = np.quantile(reference, np.linspace(0, 1, n_bins + 1))
       breakpoints[0] = -np.inf
       breakpoints[-1] = np.inf
       ref_counts = np.histogram(reference, bins=breakpoints)[0]
       cur_counts = np.histogram(current, bins=breakpoints)[0]
       # Proportions with clipping to avoid log(0)
       ref_pct = np.clip(ref_counts / ref_counts.sum(), 1e-6, None)
       cur_pct = np.clip(cur_counts / cur_counts.sum(), 1e-6, None)
       return float(np.sum((cur_pct - ref_pct) * np.log(cur_pct / ref_pct)))
   ```
