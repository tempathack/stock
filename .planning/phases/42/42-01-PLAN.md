<objective>
Create the StackingEnsemble class in ml/models/ensemble.py that wraps sklearn's StackingRegressor
with a Ridge meta-learner. The class selects the top-N base models by OOS RMSE from the training
results, builds a stacking ensemble, and provides fit/predict/evaluate methods compatible with
the existing TrainingResult/ModelRegistry infrastructure.

Purpose: ADVML-01 (StackingRegressor ensemble with Ridge meta-learner from top-N base models).
Output: A standalone, tested class that can be imported and used by the training pipeline (plan 02).
The ensemble treats already-fitted pipelines as base estimators and trains Ridge on their
out-of-sample predictions to learn optimal combination weights.
</objective>

- Ensemble-specific SHAP (existing SHAP infrastructure covers individual models)
- Weighted average blending (StackingRegressor provides the stacking approach)
</scope_exclusions>
