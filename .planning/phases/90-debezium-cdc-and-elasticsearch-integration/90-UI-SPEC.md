## Layout Specification

### Page Structure

```
Container maxWidth="xl"
  PageHeader title="Search" subtitle="Search predictions, models, drift events, and stocks"
  Paper sx={{ p: 3, mb: 3 }}         ← search box card
    TextField fullWidth               ← unified search input
  Paper sx={{ p: 0 }}                ← tabs + results card
    Tabs                              ← 4 tabs with result counts
    Divider
    TabPanel                          ← active tab content
      DataGrid                        ← paginated results table
```

### Search Box

- `TextField` with `variant="outlined"`, `fullWidth`, `size="medium"`
- `InputAdornment position="start"`: `SearchIcon` in muted color `#6B60A8`
- `InputAdornment position="end"`: `CircularProgress size={16}` when `isFetching`, hidden otherwise
- Placeholder text: `"Search predictions, models, drift events, and stocks..."`
- `autoFocus` on page mount
- Debounce: 300ms before firing query
- Border focus ring uses `theme.palette.primary.main` (`#7C3AED`) — MUI default behavior with outlined variant

### Tabs

- Use MUI `Tabs` with `variant="scrollable"` and `scrollButtons="auto"` for responsive overflow
- Each tab label: `"{Entity} ({count})"` e.g., `"Predictions (142)"` — count shows total results for current query
- Active tab underline: `indicatorColor="primary"` (inherits `#7C3AED` → `#00F5FF` gradient via sx override matching TopNav pattern)
- Tab panels are NOT mounted until first activated (use `keepMounted={false}`)

### DataGrid per Tab

All tabs use `DataGrid` with these shared props:
- `density="compact"`
- `disableRowSelectionOnClick`
- `pageSizeOptions={[25, 50, 100]}`
- `initialState={{ pagination: { paginationModel: { pageSize: 50 } } }}`
- `sx={{ border: 'none', '& .MuiDataGrid-columnHeaders': { backgroundColor: 'rgba(7, 4, 26, 0.9)' } }}`

## Interaction States

All states must be implemented per tab, not globally.

| State | Trigger | Visual |
|-------|---------|--------|
| Initial / no query | Page load, empty input | Empty state (start searching) centered in tab panel, 120px icon, heading + body |
| Loading | Query debounce fires, fetch in-flight | `Skeleton` rows (8x 48px rectangles) in tab panel + `CircularProgress` in search input end adornment |
| Results | Fetch success, data present | `DataGrid` with compact density, no border |
| Empty results | Fetch success, 0 rows | Empty state (no results found) centered, heading + query echoed in body |
| Error | Fetch error | `ErrorFallback` component centered in tab panel with "Retry" button calling `refetch()` |
| Tab switch | User clicks different tab | New tab panel renders; triggers new search for that entity if not yet fetched |
| Row hover | Mouse over DataGrid row | MUI DataGrid default hover (inherits `rgba(124,58,237,0.08)` from theme) |
