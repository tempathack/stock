   ```
   kubectl wait --for=condition=Ready node --all --timeout=120s
   ```

6. **Addon idempotent enable** (handles re-run case where cluster was running but addons not enabled):
   ```
   for addon in ingress metrics-server dashboard; do
     echo "Ensuring addon enabled: $addon"
     minikube addons enable "$addon"
   done
   ```

7. **Apply namespaces:**
   ```
   kubectl apply -f "$PROJECT_ROOT/k8s/namespaces.yaml"
   ```
   - Print echo before ("Applying namespace manifests...") and after ("Namespaces applied")

8. **Verification output:**
   ```
   kubectl get namespaces
   ```
   - Final echo: "Cluster ready"

Every major step must have an echo statement before it (verbose — no silent running, per locked decision). Use plain echo with no colour codes. Use `===` or `---` separators for section clarity if desired.
