  <action>
Add to `ml/features/indicators.py`:

**1. `compute_obv(df)`:**
- sign = np.sign(close.diff())
- obv = (sign * volume).cumsum()

**2. `compute_vwap(df)`:**
- typical_price = (high + low + close) / 3
- vwap = cumsum(typical_price * volume) / cumsum(volume)

**3. `compute_volume_sma(df, period=20)`:**
- volume_sma = volume.rolling(period).mean()

**4. `compute_ad_line(df)`:**
- clv = ((close - low) - (high - close)) / (high - low)
- ad_line = (clv * volume).cumsum()

**5. `compute_returns(df, periods=[1, 5, 21])`:**
- For each period: return_{p}d = close.pct_change(p)
- For each period: log_return_{p}d = np.log(close / close.shift(p))

**6. `compute_all_indicators(df)`:**
- Call all 14 compute_* functions in order
- Return the complete DataFrame with all indicator columns
