  <action>
Add to `ml/features/indicators.py`:

**1. `_true_range(df) -> pd.Series`** (private helper):
- high_low = high - low
- high_close = abs(high - close.shift(1))
- low_close = abs(low - close.shift(1))
- return max of the three per row

**2. `compute_sma(df, periods=[20, 50, 200])`:**
- For each period: df[f"sma_{p}"] = close.rolling(p).mean()

**3. `compute_ema(df, periods=[12, 26])`:**
- For each period: df[f"ema_{p}"] = close.ewm(span=p, adjust=False).mean()

**4. `compute_adx(df, period=14)`:**
- +DM = max(high.diff(), 0) when high.diff() > -low.diff(); else 0
- -DM = max(-low.diff(), 0) when -low.diff() > high.diff(); else 0
- Smooth +DM, -DM, TR with EWM(alpha=1/period)
- +DI = 100 × smoothed_+DM / smoothed_TR
- -DI = 100 × smoothed_-DM / smoothed_TR
- DX = abs(+DI - -DI) / (+DI + -DI) × 100
- ADX = EWM(DX, alpha=1/period)
  </action>
