## Design Decisions
- Endpoints read from **file-based** registry (no DB) — matches Phases 20–22 architecture
- `/predict/{ticker}` and `/predict/bulk` read cached predictions from `latest.json` (generated by predictor component)
- `/models/comparison` scans `{registry_dir}/{model_key}/v{N}/metadata.json`, sorted by OOS RMSE ascending
- `/models/drift` reads JSONL log, returns newest first with `any_recent_drift` computed from `is_drifted` flags
- No live prediction on-demand — pre-computed predictions only (training pipeline generates them)
- Case-insensitive ticker lookup in `/predict/{ticker}`
- ConfigDict `protected_namespaces=()` on Pydantic schemas to allow `model_name` field
