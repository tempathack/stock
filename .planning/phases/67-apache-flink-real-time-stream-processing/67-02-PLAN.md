```python
from feast import FeatureStore
from feast.push_source import PushMode
store = FeatureStore(repo_path="/opt/feast")

def push_batch_to_feast(records: list[dict]):
    if not records:
        return
    df = pd.DataFrame(records)
    df["event_timestamp"] = pd.to_datetime(df["timestamp"], utc=True)
    store.push(
        push_source_name="technical_indicators_push",
        df=df[["ticker", "event_timestamp", "ema_20", "rsi_14", "macd_signal"]],
        to=PushMode.ONLINE,
    )
```

From 67-RESEARCH.md — RSI/MACD UDAF approach:
- Use Python UDAF extending AggregateFunction from pyflink.table.udf
- The UDAF accumulates close prices over the HOP window and computes RSI/EMA on retract/flush
- RSI UDAF: accumulate list of close prices; on get_value compute RSI-14 using same Wilder's smoothing (EWM alpha=1/14) as ml/features/indicators.py Phase 10 implementation
- EMA UDAF: accumulate list of close prices; on get_value compute EMA-20 via ewm(span=20, adjust=False).mean().iloc[-1]
- MACD signal UDAF: compute MACD line (EMA12 - EMA26) then EMA-9 of MACD line series; return last value

From 67-RESEARCH.md — HOP window DDL:
```sql
-- 5-minute slide over 20-minute window (HOP window TVF syntax, Flink 1.16+)
SELECT ticker, window_start, window_end,
    rsi_udaf(close) AS rsi_14,
    ema_udaf(close) AS ema_20,
    macd_signal_udaf(close) AS macd_signal
FROM TABLE(
    HOP(TABLE intraday_source, DESCRIPTOR(`timestamp`),
        INTERVAL '5' MINUTE, INTERVAL '20' MINUTE)
)
GROUP BY ticker, window_start, window_end
```
