<objective>
Remove the hardcoded Grafana admin/admin credentials from all config files and replace
with a proper secrets pattern: K8s Secret for Kubernetes, env var substitution for
docker-compose.

Purpose: INFRA-08 requires zero hardcoded secrets. The current grafana-deployment.yaml
has `value: "admin"` and docker-compose.yml has `GF_SECURITY_ADMIN_PASSWORD=admin`.
Both are security violations.

Output:
- grafana-secret.yaml (new K8s Secret with base64-encoded placeholder password)
- grafana-deployment.yaml updated to use secretKeyRef
- docker-compose.yml updated to use ${GRAFANA_ADMIN_PASSWORD} without a default
- deploy-all.sh updated to apply the Secret before the Deployment
- .env.example updated with GRAFANA_ADMIN_PASSWORD=changeme_grafana_local_dev_only
</objective>
